Emacs is an awesome productivity tool (with basic text editing functionality). Emacs is more keyboard focused and extensible than any other software I have encountered so far. Please take a look at my Computer Configuration to see, how I am using Emacs.

Org Mode

Org Mode is one of the most powerful build-in features of Emacs. Together with additional packages, like Org Roam, this is a perfect note taking system.
